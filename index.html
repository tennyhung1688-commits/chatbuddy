常见问题

需要一直开着电脑吗？

是的，ChatBuddy 是桌面软件，需要在 Windows 电脑上运行。但可以最小化到系统托盘，后台静默运行。

WhatsApp 会不会封号？

我们使用 WhatsApp Web 协议，模拟真人操作节奏（回复延迟、偶尔不回复等），大大降低风险。但任何第三方工具都有一定风险，建议用小号使用。

可以自定义人物设定吗？

可以！在设置面板中可以配置名字、年龄、性格、职业、兴趣爱好等。WhatsApp 和 LINE 可以分别设置不同的人物。

支持 Mac 吗？

支持！提供 macOS Apple Silicon（M1/M2/M3/M4）版本。Windows 和 macOS 均可下载使用。

购买后怎么获取激活码？

购买后请联系客服 Telegram/WhatsApp/邮件，我们会在 24 小时内发送激活码。激活码一机一码，绑定你的电脑。

怎么设置定时消息？

打开设置面板 →「定时消息」，设置时间和内容，AI 会每天自动发送。适合早安问候、晚安等场景。

不想回复某些人怎么办？

在设置面板 →「黑名单」中添加号码，这些人的消息将被自动忽略，不会触发 AI 回复。

WhatsApp 扫码后没反应？

关闭后重新启动服务试试。确保网络正常，杀毒软件可能拦截，请添加信任后重试。

换电脑了怎么办？

联系客服，提供新电脑的机器码（设置面板→会员页面可查看），我们会帮你重新绑定激活码。

LINE 自动回复怎么设置？

1) 在 Chrome/Edge 安装 LINE 扩展并扫码登录；2) 软件中点「启动 LINE」自动连接浏览器 → 浏览器中打开 LINE 扩展窗口 → 点「我已登录」即可。支持 Chrome 和 Edge 浏览器，软件会自动检测。

LINE 和 WhatsApp 能用不同人设吗？

可以！WhatsApp 和 LINE 有各自独立的人物设定页面，可以设置完全不同的名字、年龄、性格、语言等，互不影响。
